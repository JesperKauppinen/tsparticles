== Kuvaus ==

Virallinen tsParticlesin WordPress Lisäosa

Luo helposti erittäin muokattavia hiukkas-, konfetti- ja ilotulitusanimaatioita ja käytä niitä animoituina taustoina verkkosivustollasi.
Käyttövalmiita komponentteja saatavilla myös seuraaaville alustoille: Web Components, Vue.js (2.x and 3.x), Angular, Svelte, jQuery, Preact, React, Riot.js, Solid.js, Inferno.

Virallinen tsParticlesin verkkosivu with options editor: <https://particles.js.org>

== Asennus ==

1. Lataa lisäosan tiedostot `/wp-content/plugins/wordpress-particles` kansioon, tai asenna lisäosa suoraan WordPress lisäosista.
2. Aktivoi lisäosa 'Lisäosat' välilehdeltä WordPressissä.
